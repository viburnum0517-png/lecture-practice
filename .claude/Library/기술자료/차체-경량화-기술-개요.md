# 차체 경량화 기술 개요

> 자동차 차체의 무게를 줄이면서 강성·안전성을 유지하는 핵심 기술.
> 전동화(EV) 전환으로 배터리 무게 상쇄 필요성이 커지면서 더욱 중요해지고 있다.

## 핵심 경량화 전략 3가지

### 1. 소재 교체 (Material Substitution)

| 소재 | 특징 | 적용 부위 |
|------|------|-----------|
| 고장력 강판 (AHSS/UHSS) | 두께 줄이면서 강도 유지 | 사이드실, 필러, 루프 |
| 알루미늄 합금 | 철 대비 약 65% 경량 | 후드, 도어, 서브프레임 |
| 탄소섬유 (CFRP) | 최고 강도·최저 무게 | 루프, 범퍼 빔 (고급차) |
| 마그네슘 합금 | 알루미늄보다 가볍지만 성형 어려움 | 시트 프레임, 대시보드 |

### 2. 구조 최적화 (Topology Optimization)

- **위상 최적화**: CAE로 하중 경로 분석 → 불필요한 재료 제거
- **두께 최적화**: 응력 집중 부위만 두껍게, 저응력 부위는 얇게
- **형상 최적화**: 단면 형상 변경으로 굽힘 강성 향상
- **적용 도구**: NASTRAN, ABAQUS, OptiStruct, HyperMesh

### 3. 공법 혁신 (Manufacturing Innovation)

- **핫 스탬핑 (Hot Stamping)**: 초고장력 강판 성형 — 복잡한 형상 + 고강도 동시 확보
- **테일러드 블랭크 (Tailored Blank)**: 부위별 두께·소재 다른 블랭크 용접 후 프레스
- **하이드로포밍**: 수압으로 복잡한 단면 형상 성형

## CAE와 경량화의 관계

```
설계 변경 제안
    ↓
CAE 해석 (강성/충돌/피로)
    ↓
목표 미달? → 소재·두께·형상 재조정
    ↓
목표 달성 → 시제품 제작
```

> **CAE 없이 경량화는 불가능**하다.
> 소재를 바꾸거나 두께를 줄일 때마다 강성·충돌 성능을 CAE로 검증해야 한다.

## 국내외 주요 경량화 성과 사례

| 제조사 | 기술 | 효과 |
|--------|------|------|
| 현대/기아 | 3세대 플랫폼 (N3) | 차체 중량 8~10% 감소 |
| BMW | CFRP 차체 (i3/i시리즈) | 차체 250kg → 130kg |
| 아우디 | 알루미늄 스페이스프레임 | 차체 40% 경량화 |
| 포드 F-150 | 전 차체 알루미늄화 | 전체 차중 320kg 감소 |

## 트렌드: AI + 경량화

- **AI 위상 최적화**: 기존 CAE보다 100배 빠른 형상 탐색 (Nvidia Modulus 등)
- **생성형 설계 (Generative Design)**: AI가 수천 개 형상 후보 자동 생성·평가
- **디지털 트윈**: 실물 없이 소재 변경 효과 가상 검증
