# DeepMind AlphaFold 소개

> Google DeepMind의 AI 기반 단백질 구조 예측 시스템. 50년간의 난제를 해결하고 2024년 노벨 화학상을 수상했다.

## 핵심 성과

| 항목 | 내용 |
|------|------|
| 출시 | 2020년 (AlphaFold2) |
| 핵심 돌파 | 단백질 폴딩 문제 50년 만에 해결 |
| 예측 정확도 | 원자 단위 정확도 |
| 분석 속도 | 수년 → **분 단위**로 단축 |
| 데이터베이스 | 2억 개 이상 단백질 구조 무료 공개 |
| 노벨상 | 2024년 화학상 공동 수상 |

## 규모

- **3백만 명** 이상의 연구자가 **190개국**에서 활용
- AlphaFold Protein Structure Database — 무료 오픈 액세스

## 활용 분야

- 질병 연구 (말라리아, 파킨슨병, 심장질환)
- 항생제 내성 대응
- 의약품 개발 가속화
- 바이오플라스틱 개발
- 식량안보 (작물 질병 대응)

## AI가 과학을 바꾸는 방식 — 핵심 인사이트

> AlphaFold는 AI가 단순 도구를 넘어 **과학적 발견의 주체**가 될 수 있음을 증명했다.
> 제조업에서도 AI가 시뮬레이션 정확도·속도를 혁신할 가능성을 보여주는 사례.

## Evan-Park 업무 연결

> CAE/시뮬레이션에서 AI가 수치 해석 속도를 AlphaFold처럼 단축하는 방향이 연구되고 있음.
> AI 기반 구조 해석 자동화 트렌드 참고 자료.
