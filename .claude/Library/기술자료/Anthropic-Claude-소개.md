# Anthropic Claude 소개

> Anthropic이 만든 AI 어시스턴트. 작문·코드·분석·리서치·파일처리를 지원한다.

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 작문 및 콘텐츠 제작 | 글쓰기, 아이디어 브레인스토밍, 음성 개발 |
| 코드 작성 및 디버깅 | 자연어 명령으로 프로그래밍 작업 처리 |
| 데이터 분석 및 시각화 | 파일 업로드로 패턴 식별·인사이트 도출 |
| 웹 검색 및 리서치 | 최신 정보 수집, Google Workspace 연동 |
| 파일 처리 및 협업 | PDF, Excel, PowerPoint 등 다양한 문서 분석 |

## 모델 종류

- **Opus** — 가장 강력한 모델 (복잡한 분석·추론)
- **Sonnet** — 균형형 모델 (일반 업무 최적)
- **Haiku** — 빠른 처리 (단순 반복 작업)

## 활용 분야

- 소프트웨어 개발 및 기술 구현
- 비즈니스 전략 및 컨설팅
- 데이터 분석 (CSV, Excel → 인사이트)
- 고객 지원, 의료·금융 엔터프라이즈

## Evan-Park 업무 연결

> CAE 결과 파일(Excel/PDF)을 업로드해서 핵심 수치 요약, 설계 검토 메모 자동 작성에 활용 가능.
