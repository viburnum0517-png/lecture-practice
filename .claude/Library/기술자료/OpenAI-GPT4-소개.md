# OpenAI GPT-4 소개

> OpenAI의 대규모 멀티모달 언어 모델. 텍스트와 이미지를 동시에 처리할 수 있는 첫 GPT 시리즈.

## 핵심 특징

| 항목 | 내용 |
|------|------|
| 출시 | 2023년 3월 |
| 입력 유형 | 텍스트 + 이미지 (멀티모달) |
| 주요 개선 | GPT-3.5 대비 정확도·추론 능력 대폭 향상 |
| 컨텍스트 | 최대 128K 토큰 (GPT-4 Turbo 기준) |
| 안전성 | RLHF + Constitutional AI 방식으로 위험 출력 감소 |

## 성능 벤치마크

- **변호사 시험(Bar Exam)**: 상위 10% 수준
- **SAT 수학**: 700/800점
- **의사 면허 시험(USMLE)**: 합격 수준
- 다양한 전문 분야 시험에서 GPT-3.5 대비 압도적 성능

## 활용 분야

- 복잡한 문서 분석 및 요약
- 이미지 내용 분석 (CAD 도면, 차트, 그래프 해석)
- 코드 생성 및 버그 수정
- 다국어 번역 및 문서 작성
- 연구·보고서 초안 작성

## Claude vs GPT-4 비교

| 항목 | Claude (Anthropic) | GPT-4 (OpenAI) |
|------|-------------------|----------------|
| 강점 | 긴 문서 처리, 안전성 | 멀티모달, 범용성 |
| 컨텍스트 | 200K 토큰 | 128K 토큰 |
| 특화 | 분석·추론 | 이미지 이해 |

## Evan-Park 업무 연결

> 설계 도면 이미지나 CAE 결과 스크린샷을 GPT-4에 올려서 내용 해석 요청 가능.
