## 폴더 구조

```
.claude/
├── Mine/              ← 내가 직접 작성한 노트
│   ├── 설계분석/      ← CAE, 시뮬레이션 결과 분석 메모
│   ├── 설계검토/      ← 설계 검토 회의 내용, 체크리스트
│   └── 용역관리/      ← 설계용역 업체, 진행상황 메모
├── Library/           ← 외부 자료, 참고 문서 보관
│   ├── 기술자료/      ← 기술 규격, 설계 기준 문서
│   └── 용역사/        ← 용역사 관련 자료
├── Inbox/             ← 아직 정리 안 된 자료 임시 보관
└── NAVIGATION.md      ← 지금 이 파일
```
