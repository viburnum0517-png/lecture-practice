> "Git은 어렵지 않아요. '저장해줘', '올려줘' 두 마디면 됩니다. Google Drive처럼 자동 동기화가 아니라, 직접 저장하고 직접 올려야 하는 차이가 있어요. 그게 오히려 더 안전해요."

| Git 용어 | 쉬운 말 | 비유 |
|---------|--------|------|
| Repository | 프로젝트 저장소 | GitHub의 공유 폴더 |
| Commit | 저장하기 | 파일을 포장해서 라벨 붙이기 |
| Push | 올리기 | 택배로 GitHub에 보내기 |
| Pull Request | 검토 요청 | Google Docs 수정 제안 모드 |

  → github.com → 내 저장소 → 방금 올린 파일 확인
