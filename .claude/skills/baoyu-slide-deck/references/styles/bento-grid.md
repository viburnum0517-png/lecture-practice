# bento-grid

Japanese lunch box inspired modular grid layout with organized rectangular sections

## Design Aesthetic

Information architecture inspired by Japanese bento boxes - distinct compartments that organize content beautifully. Each cell is self-contained yet contributes to the whole. Modern web design trend popularized by Apple's feature pages. Clean, organized, and visually satisfying.

## Background

- Color: Light Gray (#F5F5F7) or Pure White (#FFFFFF)
- Texture: None - clean solid for grid contrast

## Typography

### Primary Font (Headlines)

Modern geometric sans-serif like SF Pro, Inter, or Geist. Semi-bold for cell headers. Compact tracking for dense layouts.

### Secondary Font (Body)

Same family in regular weight. Small but highly legible. Optimized for information density.

## Visual Elements

- Asymmetric grid layouts (varying cell sizes)
- Rounded corner cards (12-20px radius)
- Subtle shadows or borders on cells
- Icon + text combinations in cells
- Large hero cells for key features
- Small metric cells for data points
- Consistent gap spacing (16-24px)
- Visual icons in each cell

## Style Rules

### Do

- Vary cell sizes for visual hierarchy
- Keep consistent gap spacing
- Use one accent color per cell
- Include visual elements in each cell
- Create clear groupings of related cells
- Align to an underlying grid system

### Don't

- Make all cells the same size
- Overcrowd cells with content
- Use inconsistent border radius
- Mix too many accent colors per slide
- Forget whitespace within cells
- Add slide numbers, footers, or logos

## Best For

Feature showcases, product comparisons, dashboard overviews, capability matrices, service offerings, portfolio grids, pricing tables
