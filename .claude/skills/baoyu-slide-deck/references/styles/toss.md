# toss

Korean fintech Toss-inspired clean design with signature blue accents and 3D glossy icons

## Design Aesthetic

Extreme simplicity and clarity. Toss Blue (#0064FF) as the signature accent color. Premium mobile app UI feel with subtle 3D elements and soft shadows. Everything is designed for instant comprehension - if a 5-year-old can't understand it in 3 seconds, simplify further.

## Background

- Color: Pure White (#FFFFFF)
- Texture: None - pristine clean

## Typography

### Primary Font (Headlines)

Toss Product Sans, Pretendard, or SF Pro. Medium weight. Korean-optimized for clean rendering.

### Secondary Font (Body)

Same family in regular weight. Generous letter-spacing for readability.

## Visual Elements

- 3D glossy icons with soft reflections
- Floating cards with subtle shadows
- Minimal line icons (stroke: 1.5-2px)
- Large touch-friendly tap targets
- Progress indicators and status badges
- Clean numerical displays
- Single-action focused layouts

## Style Rules

### Do

- Keep layouts extremely simple
- Use Toss Blue sparingly for maximum impact
- Create clear visual hierarchy
- Design for mobile-first thinking
- Use generous whitespace
- Make every element purposeful

### Don't

- Add decorative elements
- Use more than 3 colors per slide
- Create complex multi-column layouts
- Use small or hard-to-read text
- Add gradients or patterns
- Add slide numbers, footers, or logos

## Best For

Fintech products, mobile app presentations, Korean market, B2C services, onboarding flows, product demos, investor pitches
