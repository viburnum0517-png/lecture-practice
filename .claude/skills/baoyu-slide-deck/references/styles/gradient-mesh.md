# gradient-mesh

Dreamy flowing colors with organic mesh gradients creating ethereal abstract backgrounds

## Design Aesthetic

Liquid, flowing colors that blend organically like paint in water. Abstract and artistic, creating mood and emotion through color rather than explicit imagery. Modern and creative, inspired by album covers and contemporary digital art. Perfect for conveying innovation and creativity.

## Background

- Color: Dynamic mesh gradient (varies by mood)
- Texture: Smooth organic color transitions with subtle noise

## Typography

### Primary Font (Headlines)

Clean modern sans-serif like Inter, Satoshi, or Outfit. Medium to semi-bold weight. White or dark based on background luminosity.

### Secondary Font (Body)

Same family in regular weight. Positioned in areas with sufficient contrast.

## Visual Elements

- Large organic gradient blobs
- Flowing, liquid-like color transitions
- Subtle grain/noise overlay for texture
- Glassmorphic text containers when needed
- Minimal geometric line accents
- Floating abstract shapes
- Light leak effects
- Smooth color interpolation

## Style Rules

### Do

- Create smooth, organic color flows
- Use complementary color harmonies
- Position text in readable areas
- Add subtle noise for print-like quality
- Let gradients be the star
- Use text containers when needed for contrast

### Don't

- Create jarring color transitions
- Place text on busy gradient areas
- Use too many distinct colors (3-4 max)
- Add competing visual elements
- Make gradients too symmetrical
- Add slide numbers, footers, or logos

## Best For

Creative agencies, music and entertainment, brand launches, artistic portfolios, mood boards, abstract concepts, innovation presentations
