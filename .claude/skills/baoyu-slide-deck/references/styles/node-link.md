# node-link

Data visualization aesthetic with interconnected nodes showing knowledge relationships

## Design Aesthetic

Visualizes connections and relationships through nodes and linking lines. Inspired by knowledge graphs, neural networks, and constellation maps. Perfect for showing how concepts relate to each other. Scientific yet visually compelling.

## Background

- Color: Deep Navy (#0A1929) or Pure White (#FFFFFF)
- Texture: Subtle dot grid or constellation pattern

## Typography

### Primary Font (Headlines)

Clean technical sans-serif like Inter, IBM Plex Sans, or Source Sans Pro. Medium weight for node labels.

### Secondary Font (Body)

Same family in regular weight. Small but legible for annotations.

## Visual Elements

- Circular nodes of varying sizes
- Connecting lines with varying thickness
- Animated pulse effects on key nodes
- Gradient lines showing flow direction
- Cluster groupings for related concepts
- Hub-and-spoke patterns
- Network topology visualizations
- Data flow indicators

## Style Rules

### Do

- Size nodes by importance
- Vary line thickness by connection strength
- Create clear visual hierarchy
- Use color to categorize node types
- Group related nodes spatially
- Add subtle glow to highlight paths

### Don't

- Create tangled, unreadable graphs
- Use too many node colors
- Make all nodes the same size
- Ignore visual flow and direction
- Add slide numbers, footers, or logos

## Best For

Knowledge management, AI/ML concepts, system architecture, research papers, data relationships, network diagrams, concept mapping
