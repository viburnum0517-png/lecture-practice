# glassmorphism

Frosted glass aesthetic with translucent layers and depth through blur effects

## Design Aesthetic

Layered translucent surfaces that create depth and dimension. Inspired by iOS and macOS interfaces - frosted glass panels floating over colorful backgrounds. Modern, sophisticated, and premium. The blur effect creates visual hierarchy while maintaining context awareness.

## Background

- Color: Gradient mesh (Purple #667EEA to Pink #F093FB to Blue #4FACFE)
- Texture: Soft, organic gradient blobs behind glass panels

## Typography

### Primary Font (Headlines)

Clean sans-serif like SF Pro, Inter, or Poppins. Semi-bold weight for readability against translucent backgrounds. White or dark based on underlying colors.

### Secondary Font (Body)

Same family in regular weight. Slightly larger than usual for legibility through glass effects.

## Visual Elements

- Frosted glass cards (backdrop-blur: 20px)
- 1px white semi-transparent borders
- Subtle inner glow on glass elements
- Floating 3D icons with soft shadows
- Layered depth with multiple glass panels
- Organic gradient blobs in background
- Rounded corners (16-24px radius)
- Soft drop shadows on elevated elements

## Style Rules

### Do

- Layer multiple glass panels for depth
- Use consistent blur amount (16-24px)
- Add subtle borders to define glass edges
- Create contrast between glass and background
- Use white/light icons on glass surfaces
- Maintain readability with proper contrast

### Don't

- Use glass effect on everything
- Create too many overlapping layers
- Forget about text readability
- Use harsh or solid backgrounds
- Apply glass to small elements
- Add slide numbers, footers, or logos

## Best For

iOS/macOS app presentations, fintech products, modern SaaS demos, premium service pitches, UI/UX showcases, tech product launches
