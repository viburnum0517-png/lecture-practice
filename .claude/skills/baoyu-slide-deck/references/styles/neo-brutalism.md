# neo-brutalism

Bold, edgy design with thick black outlines and high-contrast primary colors

## Design Aesthetic

Raw, unpolished energy that demands attention. Thick black borders, vibrant primary colors, and intentionally "ugly" elements that create visual tension. Anti-corporate, rebellious, and unapologetically bold. Every element feels like it's breaking design rules on purpose.

## Background

- Color: Pure White (#FFFFFF) or Bright Yellow (#FFE500)
- Texture: None - flat solid backgrounds for maximum contrast

## Typography

### Primary Font (Headlines)

Heavy, chunky sans-serif like Impact, Anton, or Black Han Sans. Extra bold weight, often with slight rotation or offset. Oversized for dramatic impact.

### Secondary Font (Body)

Monospace or basic sans-serif (Courier New, Space Mono). Regular weight, high contrast against bold headlines.

## Visual Elements

- Thick black borders (4-8px) on all elements
- Hard drop shadows (offset, no blur)
- Floating/stacked card layouts
- Intentional misalignment for visual tension
- Bold geometric shapes
- Sticker-like decorative elements
- Emoji and icons used liberally
- Underlined or struck-through text for emphasis

## Style Rules

### Do

- Use maximum contrast between elements
- Apply thick black outlines consistently
- Create visual hierarchy through size and color
- Embrace asymmetry and "imperfect" layouts
- Use hard shadows instead of soft gradients
- Mix playful elements with serious content

### Don't

- Use subtle gradients or soft shadows
- Create perfectly aligned, corporate layouts
- Use muted or pastel colors
- Add unnecessary decorative patterns
- Make everything look "polished"
- Add slide numbers, footers, or logos

## Best For

Startup pitches, creative agencies, Gen-Z audiences, tech products, social media content, disruptive brands, indie projects, portfolio presentations
