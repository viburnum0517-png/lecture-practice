# dark-tech

Dark mode high-tech aesthetic with neon accents and futuristic HUD elements

## Design Aesthetic

Professional dark interface with glowing neon accents. Inspired by sci-fi interfaces, gaming UIs, and high-tech dashboards. Data-driven and sophisticated. Creates a sense of advanced technology and expertise.

## Background

- Color: Deep Charcoal (#0D1117) or Near Black (#121212)
- Texture: Subtle grid pattern or circuit board traces

## Typography

### Primary Font (Headlines)

Technical sans-serif like JetBrains Mono, Space Grotesk, or Roboto Mono. Medium to bold weight. Slightly condensed for data density.

### Secondary Font (Body)

Same family in regular weight. High contrast for dark backgrounds.

## Visual Elements

- Glowing neon lines and borders
- HUD-style data displays
- Animated graph visualizations
- Circuit board patterns
- Holographic effects
- Data stream visualizations
- Neural network node graphics
- Scanline effects

## Style Rules

### Do

- Use neon colors sparingly for maximum impact
- Create glowing effects around key elements
- Include data visualization elements
- Maintain high contrast for readability
- Add subtle tech patterns in backgrounds
- Use monospace fonts for data

### Don't

- Overuse glow effects (causes visual fatigue)
- Mix too many neon colors at once
- Sacrifice readability for aesthetics
- Make backgrounds too busy
- Add slide numbers, footers, or logos

## Best For

Tech products, AI/ML presentations, gaming, cybersecurity, data science, developer tools, startup pitches, innovation showcases
