# one-more-thing

Steve Jobs "One More Thing" style - extreme simplicity and emotional impact

## Philosophy

The audience should never read your slides - they should listen to YOU. Each slide delivers exactly one message with maximum emotional impact. Complex analysis gets distilled to its purest essence. Less is exponentially more.

## Content Structure

**Per Slide:**
- One single sentence (the headline)
- Three keywords maximum (supporting points)
- One visual element (icon, image, or number)

**Presentation Arc:**
1. Hook - Capture attention with a bold statement
2. Problem - Create tension
3. Solution - Your product/idea as the hero
4. Impact - Show the transformation
5. "One More Thing" - Surprise finale

## Writing Style

- **Tone:** Confident, visionary, slightly theatrical
- **Length:** Ultra-short. If it can't fit in a tweet, cut it.
- **Language:**
  - Use emotional words: "magical," "incredible," "revolutionary"
  - Avoid jargon - if your grandmother doesn't understand it, rephrase
  - Speak in absolutes: "This changes everything"

## Template

```
Slide 1: [Bold claim about the future]
Slide 2: [The problem everyone ignores]
Slide 3: [The insight]
Slide 4: [Your solution - one word or phrase]
Slide 5: [The impact - one powerful number]
Slide 6: "One more thing..." [Surprise revelation]
```

## Best For

Product launches, keynote presentations, vision statements, investor pitches, TED-style talks, brand manifestos
