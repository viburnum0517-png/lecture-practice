# toss-direct

Toss-style extreme clarity - if a child can't understand it in 3 seconds, simplify

## Philosophy

Complexity is the enemy. Every sentence must pass the "초등학생 테스트" - if a elementary school student can't understand it immediately, it's too complex. Strip away all expert jargon. Get to the actionable point instantly.

## Content Structure

**Per Slide:**
- Maximum 3 bullet points
- Each bullet = one simple sentence
- Final line = "한 줄 요약" (one-line summary with action)

**Information Hierarchy:**
1. What you need to know (headline)
2. Why it matters (3 bullets max)
3. What to do next (single action)

## Writing Style

- **Tone:** Friendly, clear, empowering
- **Length:** Ultra-concise. Break complex sentences into simple ones.
- **Language:**
  - Use everyday words, not business speak
  - Short sentences (max 15 words)
  - Active voice only
  - Numbers over adjectives ("3x faster" not "much faster")

## Template

```
[간단한 제목]

• 핵심 포인트 1
• 핵심 포인트 2
• 핵심 포인트 3

→ 한 줄 요약: [즉시 실행 가능한 행동 지침]
```

## Best For

Onboarding flows, user guides, beginner tutorials, Korean audiences, mobile-first content, B2C communication, support documentation

## NotebookLM Prompt

```
모든 답변을 토스(Toss)의 디자인 철학인 '간결함'에 맞추어 재구성하세요. 전문 용어는 초등학생도 이해할 만큼 쉽게 풀이하고, 내용은 불렛포인트 3개 이내로 요약하세요. 마지막에는 '한 줄 요약'을 배치하여 사용자가 즉시 실행할 수 있는 행동 지침을 제시하세요. 복잡한 문장은 단문으로 쪼개어 가독성을 극대화하세요.
```
